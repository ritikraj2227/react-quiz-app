## Usage

### Copy This Repository  

```
git clone https://github.com/ritikraj2227/Todo_app_React
```

### Install dependencies

```
npm install
```

### Run React dev server (http://localhost:3000)

```
npm start
```

### Run the JSON server (http://localhost:5000)

```
npm run server
```
